## Findings considered and NOT turned into plans

Surfaced in the audit, deliberately not planned in this pass (recorded so they are
not re-audited from scratch):

**Downgraded (need runtime confirmation before a fix is worth writing):**
- **git-http-backend PATH_INFO traversal / tar-slip in `unpackGit`** — MED
  confidence; everything under the git cache root is Arker's own already-public
  archives, and the tar is produced by Arker's own clone, so blast radius is low.
  Confirm at runtime how far `..` traverses through `git-http-backend` before
  investing; then add a PATH_INFO whitelist + a `filepath.Rel` containment check.

**Real but lower-leverage (not in the selected bundles):**
- Archived content served same-origin without consistent iframe sandboxing (itch
  iframe has no `sandbox`; MHTML render route directly navigable) — real stored-XSS
  surface, **L effort / MED risk**; deferred because it needs a sandbox-origin or
  strict-CSP design, not a quick fix.
- Dependency updates incl. go-git `v5.8.1 → v5.19.1` (pre-CVE-fix; clones
  attacker-supplied URLs), `x/crypto`, `x/net`, gin — M effort; worth doing but
  wants a go-git smoke test and `govulncheck` (not installed) to confirm
  reachability.
- SSRF guard is entry-URL-only (redirects/DNS-rebinding/pre-check probe bypass) — M
  effort / MED risk; a real hardening gap, deferred with the XSS work.
- itch single-file serving buffers the entire ZIP into RAM per request (200MB hard
  refusal papers over it) — M effort perf fix.
- Admin dashboard preloads full `ArchiveItem` rows incl. the `Logs` text column at
  1000/page — S–M perf fix.
- API-key bcrypt 72-byte truncation for very long key names — S; low exploitability.
- No CSRF tokens on state-changing admin POSTs — M; partly mitigated once 006 sets
  SameSite=Lax.
- Login username-enumeration timing + no auth rate limiting — S.
- `QueueCapture` get-or-create race can 500 on simultaneous new-URL captures — M.
- S3 seekable reader builds a double-slashed key when `S3_PREFIX` lacks a trailing
  slash — S; latent (only on the direct-URL fallback path).
- Stale docs: README advertises removed zstd/"seekable" storage; AGENT.md lists
  phantom files/tests and omits River — S; worth a cleanup pass.
- No `.golangci.yml` (only `go vet`) — S; would catch unchecked errors.
